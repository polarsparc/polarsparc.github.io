List Docker Image Tags (from Hosted Registries)

Bhaskar S 12/28/2025

This tip is for the users of Ubuntu Linux who are trying to list available tags for a docker image hosted on one of the three registries - hub.docker.com OR ghcr.io OR quay.io:

  • 1. Install skopeo by executing the command: "sudo apt install skopeo -y"
  • 2. To list tags for the 'ollama' image on hub.docker.com, execute the comand: 'skopeo list-tags docker://docker.io/ollama/ollama'
  • 3. To list tags for the 'odot' image on ghcr.io, execute the comand: 'skopeo list-tags docker://ghcr.io/obot-platform/obot'
  • 4. To list tags for the 'debezium' image on quay.io, execute the comand: 'skopeo list-tags docker://quay.io/debezium/base'
