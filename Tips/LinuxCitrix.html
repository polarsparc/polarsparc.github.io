Citrix Workspace App for Linux Gotcha

Bhaskar S 12/05/2022

This tip is for the users of Citrix Workspace App for Linux (post installation):

  • 1. Execute the command: '/opt/Citrix/ICAClient/util/HdxRtcEngine'
  • 2. If you enconter the error: "error while loading shared libraries: libc++.so.1: cannot open shared object file: No such file or directory"
  • 3. Fix the issue by executing the comand: 'sudo apt install libc++1'
